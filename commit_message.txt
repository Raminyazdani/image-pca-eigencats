Implemented eigencats_svd() function using singular value decomposition.

Function features:
- Uses numpy.linalg.svd for decomposition
- Extracts right singular vectors as eigencats
- Returns singular values and eigencats

SVD provides an alternative, often more numerically stable approach
to PCA. The right singular vectors of the centered data matrix are
equivalent to the eigenvectors of the covariance matrix.

Why: SVD is more efficient and stable for PCA than EVD.
Verified: Function computes eigencats using SVD approach.
