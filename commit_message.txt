Implemented eigencats_evd() function using eigenvalue decomposition.

Function features:
- Computes covariance matrix from centered data
- Performs eigendecomposition using numpy.linalg.eig
- Returns sorted eigenvalues and eigenvectors (eigencats)

This is the classic PCA approach using EVD. The eigenvectors of the
covariance matrix are the principal components, ordered by their
corresponding eigenvalues (variance explained).

Why: EVD is the fundamental mathematical approach to PCA.
Verified: Function correctly computes and sorts eigencats.
