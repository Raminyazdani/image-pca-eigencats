# Image PCA & Eigencats Analysis

Principal Component Analysis on cat face images.

## Overview

Core PCA implementation using eigenvalue decomposition.

## Tech Stack

- Python 3.6+, NumPy, Matplotlib, Jupyter Notebook

## Setup

```bash
pip install -r requirements.txt
```

## Status

Basic PCA implementation complete. Extensions coming next.

## Current Features

- Data preprocessing (mean-centering, flattening)
- Eigenvalue decomposition for PCA
- Eigenvector sorting by eigenvalues
- Basic visualization helpers
